Создание токена в сети ZKsync Era (стандарт ERC-20) выполняется через развертывание смарт-контракта. Самый простой способ сделать это без глубоких навыков программирования — использовать онлайн-генераторы или среду Remix IDE со специальным плагином. Ниже представлена пошаговая инструкция для самостоятельного создания токена.  

Шаг 1. Подготовка кошелька и сети  
1. Установите расширение для браузера MetaMask.  
2. Добавьте сеть ZKsync Era в свой кошелек, воспользовавшись сервисом Chainlist (найдите сеть ZKsync Era Mainnet и нажмите Add to MetaMask).  
3. Пополните баланс кошелька в сети ZKsync Era (нативным токеном ETH). Вы можете перевести средства из сети Ethereum с помощью zkSync Bridge или использовать сторонние кросс-чейн мосты (например, Orbiter Finance).  

Шаг 2. Написание и развертывание контракта (Без программирования)  
Для быстрого создания токена с фиксированной эмиссией удобно использовать сервисы-генераторы, такие как SmartContracts Token Generator https://www.smartcontracts.tools/token-generator/create/zksync/:  
1. Перейдите на сайт генератора и выберите сеть ZKsync Era.  
2. Задайте базовые параметры вашего токена:  
Name (название — например, MyToken).  
Symbol (тикер — например, MTK).  
Decimals (количество знаков после запятой, обычно 18).  
Initial Supply (начальная эмиссия — общее количество монет).  
3. Подключите кошелек и нажмите кнопку Deploy (Развернуть).  
4. Подтвердите транзакцию в кошельке MetaMask. Учитывайте, что за развертывание контракта взимается комиссия сети (Gas) в ETH.  

Шаг 3. Добавление токена в кошелекПосле успешной транзакции вы получите адрес вашего нового токена.
1. Откройте MetaMask и перейдите на вкладку токенов.  
2. Нажмите Import tokens (Импорт токенов).  
3. Вставьте адрес смарт-контракта. Система автоматически определит название и количество, после чего токен отобразится на вашем балансе.
