MetaMaskОткройте расширение MetaMask.Нажмите на выбор сетей в левом верхнем углу.Включите переключатель «Показать тестовые сети».Выберите из списка сеть Sepolia (или Polygon Amoy).Шаг 2. Получение бесплатных тестовых монетДля записи токена в блокчейн нужен «газ» (комиссия сети). Получим его бесплатно:Перейдите на проверенный кран: Google Cloud Sepolia Faucet или Alchemy Sepolia Faucet.Скопируйте адрес своего кошелька из MetaMask (начинается на 0x...).Вставьте адрес в поле на сайте крана и нажмите «Send Me ETH». Через минуту на ваш баланс поступят тестовые монеты.Шаг 3. Подключение к Bitbond Token ToolПерейдите на страницу конструктора: Bitbond Token Tool.В правом верхнем углу нажмите кнопку «Connect Wallet».Выберите MetaMask в появившемся окне и подтвердите подключение в самом кошельке.В шапке сайта Bitbond убедитесь, что в поле выбора сети установлена именно Sepolia (рядом с вашим адресом появится пометка Testnet).Шаг 4. Настройка параметров RWA-токенаВ левом боковом меню выберите раздел «Create Token» и заполните простую форму:Token Type: Выберите Standard Token (ERC20) — базовый стандарт.Token Name: Введите полное название (например, My Test Real Estate).Token Symbol: Короткий тикер из 3–4 букв (например, MTRE).Initial Supply: Общее количество токенов (например, 1000000 для деления здания на 1 млн долей).Decimals: Оставьте значение 18 (это стандартная дробность для блокчейна).Шаг 5. Активация функций контроля (Опции RWA)Чуть ниже на странице поставьте галочки напротив функций, обязательных для RWA:Can Mint: Позволит вам довыпускать токены, если реальный актив увеличится в стоимости.Can Burn: Позволит уничтожать токены (например, при выкупе доли обратно у инвестора).Can Pause: Важнейшая функция для RWA — позволяет заморозить все переводы токенов по требованию регулятора или суда.Шаг 6. Бесплатный выпуск токенаПрокрутите страницу вниз и нажмите синюю кнопку «Create Token».Платформа Bitbond покажет окно подтверждения. Поскольку это тестовая сеть, плата за сам сервис составит 0 долларов.Всплывет окно кошелька MetaMask с запросом на подписание транзакции.Проверьте, что в графе «Плата за газ» списываются тестовые ETH (SepoliaETH), и нажмите «Подтвердить» (Confirm).Подождите 10–15 секунд, пока блокчейн обработает транзакцию.Поздравляем! На экране появится сообщение об успешном выпуске и адрес вашего смарт-контракта. Все выпущенные токены мгновенно зачислятся на ваш личный кошелек MetaMask.Хотите узнать, как скопировать адрес созданного контракта и добавить отображение вашего нового токена прямо внутрь интерфейса MetaMask?
